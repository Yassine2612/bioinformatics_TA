Learning objectives
-------------------
* You are familiar with variables and types
* You can use vectors, lists and data frames
* You are familiar with functions and can define their own
* You can control program flow
* You can read and write data to and from files
* You can plot basic figures and customise them
* You can load and install libraries to access additional functionality
