Learning objectives
-------------------
* You know how to connect to a remote server and execute commands with arguments
* You are familiar with the structure of the UNIX filesystem and how to navigate it
* You can manipulate files, filenames and directory structures. You can transfer files to and from remote servers and the internet
* You are familiar with file extensions, formats and ways to look at and parse a file
* You are able to wrangle data by combining commands and controlling output
